# Smart B-Roll Planner (Frontend)

React + Vite frontend for planning and generating B-roll insertions for UGC videos using a backend API.

## Tech
React, TypeScript, Vite, Tailwind CSS, shadcn/ui, React Router

## Setup

```bash
npm install
npm run dev
```

# Create .env from .env.example:
```bash
VITE_BACKEND_URL=http://localhost:3001
```
# Features
 - Backend health check, A-roll/B-roll input with metadata, timeline generation, video preview, toast notifications.
